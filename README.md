# desmos-pixel-art
Takes an image (maybe multiple) from a directory and gets the colours exported in RGB

# What to copy into desmos

h=64

N=\left[1...,wh\right]

X=\operatorname{mod}\left(N-1,w\right)

Y=\operatorname{floor}\left(\frac{N-1}{w}\right)

\left(X,\ -Y\right)

c= rgb(R,G,B)
