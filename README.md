# desmos-pixel-art
image to 64x64 desmos pixel art

# What to copy into desmos
w=64

h=64

N=\left[1...,wh\right]

X=\operatorname{mod}\left(N-1,w\right)

Y=\operatorname{floor}\left(\frac{N-1}{w}\right)

\left(X,\ -Y\right)

c= rgb(R,G,B)

*Paste in output*

hold left click on the colour option of (x,-y) and change to c.
